## [Fish Hook](https://github.com/facebook/fishhook)

自已的代码编译生成的函数，都存放于macho文件的`__TEXT`区  
如果是系统的原函数，则位于`__DATA`区

如果检测到系统的函数(如open,getenv等)，它的函数地址位于`__TEXT`区，则可断定它被fish hook重绑定了

### 环境: arm64 
### tested in: iOS11, iOS9
